# Vacancies

Этот проект предоставляет возможности для обработки данных о размещенных на различных сайтах вакансиях: запрос вакансий по ключевому слову, сортировка по зарплате, сохранение в файл, поиск среди сохраненных вакансий.

## Описание
В проекте представлены классы:

-   `BaseInteraction`: абстрактный класс, представляющий общую функциональность для запросов с сайтов, размещающих информацию о вакансиях.
-   `HHruInteraction`: предназначен для получения данных с сервера `hh.ru`.
-   `Vacancy`: абстрактный класс, представляющий общие методы для работы с вакансиями: сохранение, чтение из файла, добавление в файл.
-   `VacancyHH`: класс, представляющий методы для работы с вакансиями, полученными с сайта `hh.ru`: сохранение, чтение из файла, добавление в файл, формирование топа по уровню зарплаты.
-   `FileWorker`: абстрактный класс, представляющий общие методы для работы файлами: создание, обновление данных, чтение данных, записанных в файл.
-   `JsonFileWorker`: класс, представляющий методы для работы файлами формата `JSON`: создание, обновление данных, чтение данных, записанных в файл.
-   `DBManager`: класс, помогающий подключиться к базе данных с сохраненными вакансиями и произвести выборку по уровню зарплат или ключевому слову.
И отдельные функции:
-   `get_salary_value`: преобразует данные о зарплате в сравнимые значения типа "integer"(число).
-   `create_database`: автоматически создает базу данных для хранения информации о вакансиях.
-   `create_table`: автоматически создает в базе данных таблицы для хранения информации о работодателях и вакансиях.
-   `fill_data`: автоматически записывает в таблицы информацию о работодателях и вакансиях.

## Точка входа
Точкой входа является файл `main.py` в корне проекта. В нем функция `main()` позволяет вам получать различные данные и совершать операции с классами, представленными в проекте.

## Тестирование

Для тестирования работы каждой классов и их методов в условиях получения различных входных данных (в том числе, ошибочных и неполных) существует группа тестов в пакете `tests`.
В модуле `conftest.py` содержатся вспомогательные функции (фикстуры), используемые при проведении тестов.

## Установка

Для установки и запуска проекта необходимо выполнить следующие шаги:

1.  **Клонируйте репозиторий:**

    ```
    git clone https://github.com/OksanaNiklashkova/Vacancies.git
    ```

2.  **Перейдите в папку проекта:**

    ```
    cd Vacancies
    ```

3.  **Установите зависимости с помощью Poetry:**

    ```
    poetry install
    ```
4. **Запустите файл main.py, чтобы воспользоваться различными методами обработки объектов классов**
    ```
    python main.py
    ```

## Зависимости

Проект использует следующие зависимости:

*   Python 3.13
*   Poetry (для управления зависимостями)
*   requests от 2.32.3 до 3.0.0
